How to create a .jar file for hadoop in eclipse

This is based on trying to compile the examples found on the icluster.

1. Download all the files in /home/aa/projects/hadoop/hadoop/lib/*
- There is no need for what's in the subdirectories
2. Also download /home/aa/projects/hadoop/hadoop/hadoop-core-1.0.1.jar
3. In Eclipse add all the above jars as external jars
4. To create .jar in Eclipse File -> Export -> Java -> JAR file -> <project>... (resource: http://help.eclipse.org/helios/index.jsp?topic=%2Forg.eclipse.jdt.doc.user%2Ftasks%2Ftasks-33.htm)

Hope that helps. Please add if you feel this is inadequate information.

          Shiry  (4 days ago) - Note that if you need to include external jars that you use in your code, your driver needs to implement the Tool interface: http://hadoop.apache.org/common/docs/current/api/org/apache/hadoop/util/Tool.html

          So that you will be able to use the -libjars options:
          http://hadoop.apache.org/common/docs/current/api/org/apache/hadoop/util/GenericOptionsParser.html

          Like so:

          export LIBJARS="-libjars one_external_jar.jar,another_external_jar.jar"

          hadoop jar your_jar.jar YourDriverClass $LIBJARS

          If you fail to do this, you will get the following exception from your mapper classes:
          Error: java.lang.ClassNotFoundException: nameOfClassInExternalJarFile
